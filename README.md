# Robot Car with Blynk by F_Embeddedd+ Team
This project is the part of final project of Embedded Programming, Burapha University

## Project details
This project use Blynk to control RC
2 Wheel drive car

## Component use in this project
  - NodeMCU ESP8266 V3
  - L298N
  - 2x DC motor
  - 4x AA battery
  - Acrylic frame

##Required library 
  -ESP8266
  -Blynk
